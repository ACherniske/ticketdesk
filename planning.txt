Current Limitations on GitHub Enterprise Server v3.13.0:
No Support for addProjectV2ItemByTitle:
This mutation, which directly adds new items with text content (as a row in the table), is not supported in your server version.
Cannot Attach Files/Images to Draft Issues:
The API does not support attaching files or images directly to draft issues.
Workaround: Use Markdown with externally hosted file/image links in the body field.
